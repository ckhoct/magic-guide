# magic-guide

## Description

此為**懷舊專案**, `初學程式時的小專作品` <br/>
把涉及個人資料的部分移除後, 從新放上`repositories`來展示 <br/>

### 小專回憶錄

> 初學前端幾個月時, 導師會要求班上同學在兩週還是一週內完成小專作品 <br/>
> 當初的要求是: 4 頁 RWD 靜態網頁, 主題自選, 畫面純手刻, 不能套版 <br/>
> 那個時候光是想主題就覺得很燒腦了, 然後要怎麼排版完全沒概念 <br/>
> 時光荏苒, 最後還是順利完成了所有要求項目 <br/>
<br />
現在看當初的作品還是覺得很有意思 <br/>
裡面有很多小彩蛋, 還有CSS做的3D效果 <br/>
特別是能夠在還沒有什麼技術下去完成一樣作品 <br/>
<br />
雖然很有意思, <br />
但現在回頭看看, <br/>
沒有模組化, 沒有腳手架, <br/>
看到程式碼都寫在一起, 就不會想動它了XD
